### RSS aggregator

This aggregator helps you to collect RSS feeds and watches for new posts.

## Installation and setup:
```
# clone repo
git clone git@github.com:heyanny2/frontend-project-11.git

# install dependencies
make install

# install the package
npm link

# run the webpack on localhost
npm start

# webpack build
make build
```
